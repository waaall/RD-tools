## 可修改任务列表顺序

只控制“显示顺序”，不改任务本身的 key、标题、功能。
只是“排序”，不需要支持“隐藏某些任务”之类的额外功能
不用影响“设置页里的任务设置列表”
顺序配置保存为 task_key 或者 task_order 之类的，不用中文标题。
配置缺失时，给个 warning 回退到现在代码里的默认顺序。
配置里没写到的新任务，自动追加到末尾，避免升级后任务消失。
同一份顺序同时作用于“任务中心”和“设置 > 任务设置”，保持一致。
UI 长按可以拖拽排序，排序后存放在setting.json用户级别保存。
